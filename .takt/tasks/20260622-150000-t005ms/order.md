# タスク仕様

## 目的

wgpuベースのGPU計算パイプライン基盤を構築する。コンピュートシェーダのディスパッチ、テクスチャプール管理、シェーダモジュール管理を含むGPU処理の共通インフラを提供する。

## 要件

- [ ] wgpu Device/Queue初期化（アダプタ選択、デバイス要件定義）
- [ ] コンピュートシェーダディスパッチ基盤（パイプライン生成、バインドグループ管理）
- [ ] テクスチャプール実装（アロケーション、再利用、LRU解放）
- [ ] シェーダモジュール管理（コンパイル、キャッシュ、バリデーション）
- [ ] デバッグ用シェーダホットリロード機能
- [ ] GPU↔CPU転送ユーティリティ（バッファ読み戻し、テクスチャダウンロード）
- [ ] Metal（macOS）+ D3D11/D3D12（Windows）バックエンドでの動作検証

## 受け入れ基準

- 簡単なコンピュートシェーダ（画像反転等）がGPU上で正しく実行される
- テクスチャプールが同一サイズ・フォーマットのテクスチャを再利用する
- シェーダコンパイルエラーが人間可読なエラーメッセージで報告される
- ホットリロードでシェーダファイル変更後に再コンパイル・反映される
- macOS（Metal）とWindows（D3D11）の両方でテスト通過
