# タスク仕様

## 目的

GPUIフレームワークによるアプリケーションシェルを構築する。メインウィンドウ、メニューバー、ドッキング可能なパネルシステム、ワークスペースプリセット、キーバインド機構を含むUI基盤を提供する。

## 要件

- [ ] GPUIアプリケーションブートストラップ（起動シーケンス、イベントループ）
- [ ] メインウィンドウ + メニューバー構成
- [ ] ワークスペースパネルシステム（gpui_componentのdock/sheet活用）
- [ ] ワークスペースプリセット定義（Edit / Node / Color / Motion）
- [ ] Viewメニューからのパネル表示切替
- [ ] キーバインドシステム（JSON/TOMLによるカスタマイズ可能な設定）
- [ ] マルチウィンドウ対応（パネルのデタッチ・独立ウィンドウ化）
- [ ] プロパティインスペクターシェル（空パネル、後続タスクで中身実装）
- [ ] macOS + Windowsでの動作検証

## 受け入れ基準

- アプリケーション起動でメインウィンドウとメニューバーが表示される
- 4種のワークスペースプリセット切替でパネルレイアウトが変化する
- パネルのドラッグ&ドロップによるドッキング位置変更が動作する
- キーバインド設定ファイル変更が次回起動時に反映される
- パネルをデタッチして独立ウィンドウとして操作可能
- macOSとWindowsで同等のUI動作
