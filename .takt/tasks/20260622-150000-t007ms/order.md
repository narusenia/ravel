# タスク仕様

## 目的

アニメーションチャネルシステムを実装する。キーフレームカーブ、定数値、式、ノード出力参照、ブレンド等の多様なチャネルソースによるパラメータアニメーション基盤を提供する。

## 要件

- [ ] ChannelSource enum定義（Constant, Keyframes, Expression（プレースホルダ）, NodeOutput, AudioReactive（プレースホルダ）, Blend）
- [ ] AnimationChannel評価エンジン（任意時刻でのチャネル値算出）
- [ ] KeyframeCurve実装（ベジェ補間、リニア補間、ステップ補間）
- [ ] キーフレームCRUD操作（追加、削除、更新、移動）
- [ ] ユニットテスト：ベジェカーブ補間の精度検証
- [ ] ユニットテスト：リニア・ステップ補間の境界値テスト
- [ ] ユニットテスト：Blendチャネルの合成結果検証

## 受け入れ基準

- 各ChannelSourceが正しい値を返す（定数→固定値、キーフレーム→補間値）
- ベジェ補間がAfter Effectsと同等のイージングカーブを再現する
- キーフレーム追加・削除後にカーブが正しく再計算される
- 時刻範囲外（キーフレーム前後）でのエクストラポレーション動作が定義済み
- Expressionプレースホルダがエラーではなくデフォルト値を返す
