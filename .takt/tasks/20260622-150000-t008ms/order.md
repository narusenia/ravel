# タスク仕様

## 目的

.ravprjプロジェクトファイル形式を設計・実装する。RONシリアライゼーション、Zipコンテナ、アセット参照、設定階層、バージョンマイグレーション、自動バックアップを含む永続化基盤を提供する。

## 要件

- [ ] RONシリアライゼーションによるGraph保存・復元
- [ ] manifest.jsonの読み書き（プロジェクトメタデータ、バージョン情報）
- [ ] Zipコンテナ形式での.ravprjファイル構成
- [ ] アセット参照システム（相対パス、環境変数展開）
- [ ] プロジェクト設定（TOML形式、オーバーライド階層：デフォルト→グローバル→プロジェクト→ユーザー）
- [ ] フォーマットバージョンマイグレーションチェーン（v1→v2→...の順次変換）
- [ ] .ravprj.bak自動バックアップ（保存前に前バージョンを退避）
- [ ] グローバル設定パス解決（dirs crateによるOS準拠のパス）
- [ ] ファズテスト：不正入力に対するパーサーの堅牢性検証

## 受け入れ基準

- Graph→.ravprj保存→読み込み→Graphのラウンドトリップが完全一致
- 旧バージョンの.ravprjファイルが最新バージョンにマイグレーション成功
- 破損Zipファイル投入時にパニックせず適切なエラーメッセージ返却
- .ravprj.bakが保存操作ごとに自動生成される
- macOS/Windows/Linuxで正しいグローバル設定パスが解決される
- ファズテストで10万入力を処理してもクラッシュなし
