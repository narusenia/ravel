# Ravel

タイムライン編集とプロシージャルノードグラフを統合した次世代動画編集ソフトウェア。

## ビジョン

Ravelは、Houdini/Cavalryに着想を得たプロシージャルコンテンツ生成の力を動画編集の世界に持ち込む。ノードグラフファーストのアーキテクチャで、エフェクト・合成・トランジション・タイムラインの全てがDAG（有向非巡回グラフ）として表現される。従来のタイムラインインターフェースはDAGの上に構築された制約付きビューとして機能し、初心者からプロまで段階的に使える設計。

### 主な特徴

- **ノードグラフファースト** — 全処理がDAGで表現。タイムラインはシーケンスノードの糖衣表現。
- **プロシージャルモーショングラフィックス** — シェイプ生成、リピーター、パーティクル、フィールド/フォース、数式ドリブンパラメータ。
- **プロシージャルタイポグラフィ** — 文字単位アニメーション、パス沿いテキスト、テキスト→ジオメトリ変換、3Dテキスト押し出し。リリックビデオ/MV制作に最適化。
- **オーディオリアクティブ** — FFTスペクトル解析、ビート検出、BPM同期、タイムライン上のビートマーカー。解析結果を任意パラメータに接続可能。
- **OpenFX互換** — 業界標準プラグイン（Sapphire、BorisFX等）をC/C++ Shimレイヤー経由で実行。プロセス分離で安全性確保。
- **OpenColorIO** — 業界標準カラーマネジメント。GPU LUTによるリアルタイム適用。
- **WGSLカスタムシェーダ** — GPUエフェクトをWGSLで記述。ライブプレビュー対応。
- **Luaスクリプティング** — パラメータ制御と操作自動化のためのスクリプト環境。

## 技術スタック

| コンポーネント | 技術 |
|-------------|------|
| 言語 | Rust |
| UIフレームワーク | GPUI (Zed由来) |
| GPU計算 | wgpu + プラットフォームネイティブフォールスルー |
| メディアI/O | FFmpeg (LGPL, ダイナミックリンク) + ネイティブHWデコーダ |
| カラー | OpenColorIO |
| オーディオ | CPAL + dasp + rubato |
| スクリプティング | Lua (mlua) |

## アーキテクチャ

```
UI層 (GPUI)  →  アプリケーション層  →  コアエンジン (DAG評価)
                                            ↓
                                      メディア層 (FFmpeg, CPAL, OCIO)
                                            ↓
                                      GPU層 (wgpu)
                                            ↓
                                      プラットフォーム層 (Metal / D3D11 / Vulkan)
```

詳細は [docs/specifications/architecture.md](docs/specifications/architecture.md) を参照。

## プロジェクトファイル

`.ravprj`形式 — zipコンテナの中にRON（グラフ定義）、JSON（メタデータ）、TOML（設定）を格納。テキストベースでgit管理に対応。

## プラットフォーム対応

| プラットフォーム | ステータス | GPUバックエンド |
|---------------|----------|---------------|
| macOS | 主要開発対象 | Metal |
| Windows | 同時対応（設計段階から考慮） | Direct3D 11 |
| Linux | 将来対応 | Vulkan |
