# TASK-011: オーディオエンジン (CPAL + DSP)
- **マイルストーン**: MS2 Media Pipeline
- **関連要件**: REQ-MEDIA-002
- **規模**: M
- **依存タスク**: TASK-003

## 概要
CPALによるオーディオ出力基盤と、daspを用いたマルチトラックミキサーを構築する。専用の高優先度オーディオスレッドでリアルタイム処理を行い、rubato等によるサンプルレート変換、基本エフェクト、映像との同期機構、UI用波形データ生成を提供する。

## 実装ステップ
1. CPALオーディオ出力セットアップ
2. リアルタイムオーディオスレッド（専用・高優先度）
3. daspを用いたマルチトラックミキサー
4. rubato等によるサンプルレート変換
5. 基本エフェクト（ゲイン、フェード）
6. 映像-音声同期機構
7. UI表示用波形データ生成

## 対象コンポーネント
- `crates/ravel-audio/` (新規またはオーディオ関連クレート)

## 完了条件
- [x] CPALでオーディオ出力が正常動作
- [x] 専用高優先度スレッドでリアルタイムオーディオ処理が動作
- [x] マルチトラックミキシングが正しく動作
- [x] 異なるサンプルレートのソースが自動変換される
- [x] ゲイン・フェードエフェクトが適用可能
- [x] 映像フレームとオーディオの同期機構が機能
- [x] 波形データがUI表示用に生成される
