# TASK-012: タイムライン基盤 + メディアビン

> **注**: 本タスクの Track/Clip + SequenceNode モデルは v2 で Composition/Layer
> モデルに置換された（`ravel-core/src/composition/`、TASK-019・PR #38 参照）。
> Track/Clip 実装はコードベースに存在しない。実装状況は `docs/ui-impl-status.md` が正。
- **マイルストーン**: MS2 Media Pipeline
- **関連要件**: REQ-UI-003, REQ-UI-008, REQ-CORE-008
- **規模**: L
- **依存タスク**: TASK-009, TASK-011, TASK-006

## 概要
タイムラインのTrack/Clipデータモデルとタイムラインをノードとして扱うSequenceNode、マルチシーケンス対応を実装する。また、メディアビンUIとしてインポート、フォルダ/タグ管理、サムネイルプレビュー、メタデータ表示、検索/フィルタ機能を提供する。

## 実装ステップ
1. Track/Clipデータモデル実装
2. SequenceNode実装（タイムラインをノードとして表現）
3. マルチシーケンス対応
4. タイムラインUI上でのクリップ追加/削除/移動/トリム
5. トラック管理（追加/削除/ミュート/ロック）
6. メディアビンUI（インポート、フォルダ/タグ、サムネイルプレビュー、メタデータ表示、検索/フィルタ）
7. メディアインポートワークフロー（ファイルダイアログ + D&D）

## 対象コンポーネント
- `crates/ravel-core/` (Track/Clipデータモデル, SequenceNode)
- `crates/ravel-ui/` (タイムラインUI, メディアビンUI)
