# TASK-013: 映像/音声同期再生
- **マイルストーン**: MS2 Media Pipeline
- **関連要件**: REQ-MEDIA-001, REQ-MEDIA-002
- **規模**: M
- **依存タスク**: TASK-012

## 概要
フレーム精度の再生クロックを実装し、オーディオをマスタークロックとした映像-音声同期再生を実現する。タイムラインスクラビング、トランスポートコントロール、ビューワ統合を含む。

## 実装ステップ
1. 再生クロック実装（フレーム精度）
2. 映像-音声同期実装（オーディオをマスタークロックとして使用）
3. タイムラインスクラビング（フレーム単位シーク）
4. トランスポートコントロール（再生/一時停止/停止/フレームステップ）
5. ビューワ統合（デコード済みフレームをwgpuテクスチャ経由で表示）
6. 再生パフォーマンスプロファイリング

## 対象コンポーネント
- `crates/ravel-media/` (再生クロック, 同期機構)
- `crates/ravel-ui/` (トランスポートコントロール, ビューワ)

## 完了条件
- [ ] フレーム精度の再生クロックが正しく動作
- [ ] オーディオマスタークロックで映像-音声がリップシンク
- [ ] タイムラインスクラビングでフレーム単位のシークが可能
- [ ] 再生/一時停止/停止/フレームステップが動作
- [ ] ビューワにデコード済みフレームがリアルタイム表示
- [ ] 再生パフォーマンスのプロファイリング結果を記録
