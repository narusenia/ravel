# TASK-014: ノードグラフエディタUI基盤
- **マイルストーン**: MS3 Node Editor
- **関連要件**: REQ-UI-002
- **規模**: L
- **依存タスク**: TASK-006

## 概要
GPUIカスタムビューとして2Dキャンバスを実装し、ノードグラフエディタの基盤UIを構築する。パン/ズーム操作、ノードのレンダリング（ヘッダ、ポート、インラインパラメータ）、ポートの型ベース色分け、グリッド背景、ミニマップ、サブグラフナビゲーション用パンくずリストを含む。

## 実装ステップ
1. パン/ズーム対応の2Dキャンバス実装（GPUIカスタムビュー）
2. ノードレンダリング（ヘッダ、ポート、インラインパラメータ）
3. ポートの型ベース色分け
4. キャンバスグリッドと背景描画
5. 大規模グラフ用ミニマップ
6. サブグラフナビゲーション用パンくずリスト

## 参考ライブラリ

- **gpui-flow** (https://github.com/pacifio/gpui-flow): GPUI向けのビジュアルノードエディタ。React Flow/xyflowにインスパイア。MIT License。ノードレンダリング、接続線描画、パン/ズーム等の基盤実装がある。zed git版gpui依存のため、crates.io gpui 0.2.2との互換性は要確認。フォークまたは参考実装として活用を検討。

## 対象コンポーネント
- `crates/ravel-ui/` (ノードグラフエディタUI)

- [ ] ポートが型に応じた色で描画
