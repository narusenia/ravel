# TASK-015: ノードグラフインタラクション
- **マイルストーン**: MS3 Node Editor
- **関連要件**: REQ-UI-002, REQ-UI-010
- **規模**: L
- **依存タスク**: TASK-014

## 概要
ノードグラフエディタ上でのユーザーインタラクションを実装する。ノード選択、ドラッグ移動（スナップガイド・自動整列付き）、ワイヤ接続（型互換チェック付き）、ノード検索パレット、サブグラフ作成/展開、コピペ、ファイルD&Dインポートを含む。

## 実装ステップ
1. ノード選択（クリック、矩形選択、Ctrl+クリックによる複数選択）
2. ノードドラッグ（スナップガイド・自動整列付き）
3. ワイヤドラッグ（ポート間接続、型互換性チェック）
4. ノード検索パレット（Tab起動、インクリメンタルサーチ、カテゴリフィルタ）
5. サブグラフ作成（Ctrl+Gでグループ化）と展開（ダブルクリックで内部表示）
6. ノード/ワイヤのコピー&ペースト（RONクリップボード）
7. ファイルD&Dインポート

## 対象コンポーネント
- `crates/ravel-ui/` (ノードグラフインタラクション)
