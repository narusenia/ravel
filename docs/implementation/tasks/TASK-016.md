# TASK-016: ビルトインノード実装 (基本セット)
- **マイルストーン**: MS3 Node Editor
- **関連要件**: REQ-MOGRAPH-005
- **規模**: L
- **依存タスク**: TASK-002, TASK-005

## 概要
ノードグラフで使用するビルトインノード群を実装する。ノード登録システムを構築し、合成・色補正・カラーグレーディング・フィルタ・トランスフォーム・ジェネレータ・トランジション・タイムの各カテゴリのノードをGPUシェーダベースで実装する。

## 実装ステップ
1. ノード登録システム実装
2. コンポジターノード: ブレンドモード、マージ、マスク
3. カラーノード: ブライトネス/コントラスト、色相/彩度、カーブ、レベル
4. カラーグレーディングノード: リフト/ガンマ/ゲイン、カラーホイール、HSLカーブ、LUT適用
5. フィルタノード: ガウシアンブラー、ディレクショナルブラー、シャープン
6. トランスフォームノード: 移動、回転、スケール、ミラー、タイル
7. ジェネレータノード: グラデーション、ノイズ、フラクタル、チェッカーボード
8. トランジションノード: クロスディゾルブ、ワイプ、プッシュ、スライド（20種類）
9. タイムノード: タイムリマップ、フレームブレンド、フリーズ
10. 各ノードのGPUシェーダ実装

## 対象コンポーネント
- `crates/ravel-core/` (ノード登録システム)
- `crates/ravel-nodes/` (ビルトインノード実装)
- `crates/ravel-gpu/` (GPUシェーダ)
