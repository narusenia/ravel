# TASK-017: タイムライン-ノードグラフ連携
- **マイルストーン**: MS3 Node Editor
- **関連要件**: REQ-UI-003
- **規模**: M
- **依存タスク**: TASK-015, TASK-012

## 概要
タイムラインのクリップにエフェクトスタックを持たせ、ノードグラフとの双方向連携を実現する。タイムラインUIからエフェクトスタックをノードグラフとして展開・編集し、変更をタイムラインに反映する。トランジション適用やインラインパラメータ編集もタイムラインUIから直接行える。

## 実装ステップ
1. タイムラインクリップへのエフェクトスタック実装
2. エフェクトスタック → ノードグラフ展開（「ノードグラフで開く」）
3. ノードグラフ変更のタイムラインへの反映
4. グラフに分岐がある場合の「カスタムエフェクト」表示
5. タイムラインUIからのトランジション適用
6. タイムライン上でのインラインパラメータ編集

## 対象コンポーネント
- `crates/ravel-core/` (エフェクトスタック, ノードグラフ連携ロジック)
- `crates/ravel-ui/` (タイムラインUI, ノードグラフUI連携)

## 完了条件

- [ ] 「ノードグラフで開く」でエフェクトスタックがノードグラフとして展開

- [ ] 分岐を持つグラフが「カスタムエフェクト」として表示
