# TASK-018: カーブエディタ
- **マイルストーン**: MS3 Node Editor
- **関連要件**: REQ-CORE-007
- **規模**: M
- **依存タスク**: TASK-007, TASK-014

## 概要
GPUIカスタムビューとしてカーブエディタパネルを実装する。ベジェ/リニア/ステップカーブの表示、キーフレーム編集、複数パラメータのカーブオーバーレイ、チャンネルソースタイプ切り替え、BPMグリッド表示を含む。

## 実装ステップ
1. カーブエディタパネル実装（GPUIカスタムビュー）
2. ベジェ/リニア/ステップカーブ表示
3. キーフレーム編集（追加/削除/移動、ハンドル調整）
4. 複数パラメータのカーブオーバーレイ表示
5. UIでのチャンネルソースタイプ切り替え
6. BPMグリッド表示（オーディオリアクティブワークフロー向け）

## 対象コンポーネント
- `crates/ravel-ui/` (カーブエディタパネル)
- `crates/ravel-core/` (カーブデータモデル連携)

## 完了条件
- [ ] カーブエディタパネルがGPUIカスタムビューとして動作
- [ ] ベジェ/リニア/ステップの各カーブタイプが正しく表示
- [ ] キーフレームの追加/削除/移動/ハンドル調整が動作
- [ ] 複数パラメータのカーブを重ねて表示可能
- [ ] チャンネルソースタイプの切り替えがUI上で動作
