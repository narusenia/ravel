# TASK-019: 三層キャッシュシステム
- **マイルストーン**: MS4 Rendering
- **関連要件**: REQ-CORE-006
- **規模**: L
- **依存タスク**: TASK-005, TASK-002

## 概要
VRAM → RAM → ディスクの三層キャッシュシステムを実装する。GPUテクスチャプール（LRU）、RAMキャッシュ（LRU、サイズ上限可変）、ディスクキャッシュ（.ravprj/.cache/）を階層的に管理し、DAG評価器のダーティフラグと連携してキャッシュの無効化を行う。

## 実装ステップ
1. VRAMキャッシュ実装（GPUテクスチャプール、LRUエビクション）
2. RAMキャッシュ実装（LRU、設定可能なサイズ上限）
3. ディスクキャッシュ実装（`.ravprj/.cache/`ディレクトリ、シリアライズ）
4. キャッシュ階層実装（VRAM → RAM → ディスクのエビクションチェーン）
5. キャッシュキー生成（ノードID + フレーム + パラメータハッシュ）
6. キャッシュヒット/ミス監視と統計
7. DAG評価器ダーティフラグとの統合

## 対象コンポーネント
- `crates/ravel-core/` (キャッシュシステム)
- `crates/ravel-gpu/` (VRAMキャッシュ, テクスチャプール)

## 完了条件
- [ ] VRAMキャッシュがLRUエビクションで正しく動作
- [ ] RAMキャッシュが設定サイズ上限でLRU動作
- [ ] ディスクキャッシュが`.ravprj/.cache/`に永続化
- [ ] VRAM → RAM → ディスクのエビクションチェーンが正しく機能
- [ ] キャッシュキーがノードID+フレーム+パラメータハッシュで一意に生成
- [ ] キャッシュヒット/ミス統計が監視可能
- [ ] DAG評価器のダーティフラグ変更時にキャッシュが適切に無効化
