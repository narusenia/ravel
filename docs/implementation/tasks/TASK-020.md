# TASK-020: レンダーキュー
- **マイルストーン**: MS4 Rendering
- **関連要件**: REQ-RENDER-001
- **規模**: M
- **依存タスク**: TASK-019, TASK-002

## 概要
レンダージョブの定義・キュー管理・バックグラウンド実行を実装する。プレビュー再生とは独立したレンダー実行環境を提供し、レンダーキューUIで進捗表示・一時停止/再開/キャンセルを操作可能にする。

## 実装ステップ
1. レンダージョブ定義（出力フォーマット、範囲、設定）
2. 優先度付きジョブキュー実装
3. バックグラウンドレンダー実行（プレビューとは分離）
4. レンダーキューUI（進捗表示、一時停止/再開/キャンセル）
5. レンダー進捗レポート

## 対象コンポーネント
- `crates/ravel-core/` (レンダージョブ, キュー管理)
- `crates/ravel-ui/` (レンダーキューUI)

## 完了条件
- [ ] レンダージョブが出力フォーマット・範囲・設定を含めて定義可能
- [ ] 優先度付きキューでジョブが正しく順序管理
- [ ] バックグラウンドでレンダーが実行されプレビュー再生と干渉しない
- [ ] レンダーキューUIで進捗・一時停止/再開/キャンセルが動作
- [ ] レンダー進捗が正しくレポート
