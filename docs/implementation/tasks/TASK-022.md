# TASK-022: エクスポートパイプライン
- **マイルストーン**: MS4 Rendering
- **関連要件**: REQ-RENDER-001
- **規模**: M
- **依存タスク**: TASK-020, TASK-009

## 概要
ノードグラフからの最終出力までのエクスポートパイプラインを実装する。エクスポート設定UI、FFmpegエンコーダ統合、HWエンコーダ対応、プログレスダイアログを含み、ノードグラフ → エクスポートファイルのフルパイプライン結合テストで品質を担保する。

## 実装ステップ
1. エクスポート設定UI（コーデック、コンテナ、解像度、フレームレート、品質）
2. FFmpegエンコーダ統合（最終出力用）
3. HWエンコーダ対応（VideoToolbox/NVENC）
4. エクスポート進捗ダイアログ
5. 結合テスト: ノードグラフからエクスポートファイルまでのフルパイプライン

## 対象コンポーネント
- `crates/ravel-media/` (FFmpegエンコーダ, HWエンコーダ)
- `crates/ravel-ui/` (エクスポート設定UI, 進捗ダイアログ)

## 完了条件
- [ ] エクスポート設定UIでコーデック・コンテナ・解像度・フレームレート・品質を選択可能
- [ ] FFmpegエンコーダで最終出力が正しく生成
- [ ] VideoToolbox/NVENCによるHWエンコードが動作
- [ ] エクスポート進捗ダイアログが正しく表示
- [ ] ノードグラフ → エクスポートファイルのフルパイプライン結合テストがパス
