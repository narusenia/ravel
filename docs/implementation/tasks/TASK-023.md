# TASK-023: シェイプ生成 + リピーターノード
- **マイルストーン**: MS5 Motion Graphics
- **関連要件**: REQ-MOGRAPH-001
- **規模**: M
- **依存タスク**: TASK-016, TASK-007

## 概要
基本シェイプ生成ノード群（矩形・楕円・多角形・星形・カスタムパス）の実装。ベジェハンドルによるパス編集、リピーターノード（グリッド配列・円形配列・パス沿い配列）、GPU描画、統一アニメーションチャネル接続、スタイライズノード（グロー・シャドウ・ストローク・エンボス）、ディストーションノード（ワープ・レンズ・リプル）を提供する。

## 実装ステップ
1. シェイプジェネレータノード実装（rectangle, ellipse, polygon, star, custom path）
2. パス編集機能実装（ベジェハンドル操作）
3. リピーターノード実装（グリッド配列・円形配列・パス沿い配列）
4. GPU経由のシェイプレンダリング実装
5. 全パラメータを統一アニメーションチャネルへ接続
6. スタイライズノード実装: グロー、シャドウ、ストローク、エンボス
7. ディストーションノード実装: ワープ、レンズ、リプル

## 対象コンポーネント
- `crates/ravel-core/src/nodes/shape/` (シェイプ生成ノード)
- `crates/ravel-core/src/nodes/repeater/` (リピーターノード)
- `crates/ravel-core/src/nodes/stylize/` (スタイライズノード)
- `crates/ravel-core/src/nodes/distortion/` (ディストーションノード)
- `crates/ravel-gpu/src/shape_renderer/` (GPU描画)

- [ ] グリッド・円形・パス沿いの各リピーターノードが動作する
