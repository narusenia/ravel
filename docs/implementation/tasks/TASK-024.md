# TASK-024: パーティクルシステム + フィールド/フォース
- **マイルストーン**: MS5 Motion Graphics
- **関連要件**: REQ-MOGRAPH-002
- **規模**: L
- **依存タスク**: TASK-023

## 概要
パーティクルエミッタノード（ポイント・ライン・エリアソース）、パーティクルライフサイクル管理、フォースノード（重力・アトラクタ・ドラッグ）、フィールドノード（ノイズ・タービュランス）、パーティクル描画（ポイント・スプライト・シェイプインスタンス）を実装。GPUアクセラレーションにより10万パーティクル以上をインタラクティブレートで処理する。

## 実装ステップ
1. パーティクルエミッタノード実装（ポイント/ライン/エリアソース）
2. パーティクルライフサイクル実装（生成・加齢・消滅）
3. フォースノード実装: 重力、アトラクタ、ドラッグ
4. フィールドノード実装: ノイズフィールド、タービュランス
5. パーティクルレンダリング実装（ポイント/スプライト/シェイプインスタンス）
6. パーティクルプロパティカーブ実装（ライフタイム中のサイズ・色・不透明度）
7. GPUアクセラレーションによるパーティクルシミュレーション
8. パフォーマンス目標達成: 10万パーティクル以上をインタラクティブレートで動作

## 対象コンポーネント
- `crates/ravel-core/src/nodes/particle/` (パーティクルノード)
- `crates/ravel-core/src/nodes/force/` (フォースノード)
- `crates/ravel-core/src/nodes/field/` (フィールドノード)
- `crates/ravel-gpu/src/particle_renderer/` (パーティクルGPU描画)
- `crates/ravel-gpu/src/compute/particle_sim/` (GPUシミュレーション)

## 完了条件

- [ ] パーティクルの生成・加齢・消滅ライフサイクルが正しく機能する
- [ ] 重力・アトラクタ・ドラッグのフォースノードが動作する
- [ ] ノイズフィールド・タービュランスのフィールドノードが動作する
- [ ] ポイント・スプライト・シェイプインスタンスでのパーティクル描画が動作する
- [ ] サイズ・色・不透明度のライフタイムカーブが機能する
- [ ] GPUアクセラレーション有効時に10万パーティクル以上がインタラクティブレートで動作する
