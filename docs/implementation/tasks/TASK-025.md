# TASK-025: プロシージャルタイポグラフィエンジン
- **マイルストーン**: MS5 Motion Graphics
- **関連要件**: REQ-MOGRAPH-004
- **規模**: L
- **依存タスク**: TASK-023

## 概要
テキストレイヤーノード（swash経由のフォント選択・サイズ・色・行間）、文字単位アニメーション（イン/アウト・ウェーブ・ランダム・スタガー）、テキストオンパス、テキスト→ジオメトリ変換（zenoによるパスデータ化）、パーティクル/フィールドシステムとの統合、ノードグラフベースのテキストアニメーター、リッチテキスト（文字単位スタイリング）を実装する。

## 実装ステップ
1. テキストレイヤーノード実装（swash経由フォント選択、サイズ、色、行間）
2. 文字単位アニメーション実装（イン/アウト、ウェーブ、ランダム、スタガー）
3. テキストオンパス実装（ベジェパスに沿ったテキスト配置）
4. テキスト→ジオメトリ変換実装（zenoを用いてテキストからパスデータを生成）
5. 変換後ジオメトリをパーティクル/フィールドシステムと統合
6. テキストアニメーターノード実装（ノードグラフベースのテキストアニメーション）
7. リッチテキストサポート実装（文字単位のスタイリング）

## 対象コンポーネント
- `crates/ravel-core/src/nodes/text/` (テキストノード)
- `crates/ravel-core/src/nodes/text/animator/` (テキストアニメーター)
- `crates/ravel-core/src/nodes/text/typography/` (タイポグラフィエンジン)
- `crates/ravel-gpu/src/text_renderer/` (テキストGPU描画)

- [ ] 文字単位のイン/アウト・ウェーブ・ランダム・スタガーアニメーションが動作する
- [ ] ベジェパスに沿ったテキスト配置が動作する
- [ ] zenoによるテキスト→パスデータ変換が動作する

- [ ] 文字単位のリッチテキストスタイリングが動作する
