# TASK-026: 3D基本機能
- **マイルストーン**: MS5 Motion Graphics
- **関連要件**: REQ-MOGRAPH-003
- **規模**: L
- **依存タスク**: TASK-005, TASK-025

## 概要
3Dテキスト押し出し・ベベル、基本3Dプリミティブ（ボックス・球・シリンダー・平面）、3Dカメラノード（透視/正射影）、基本ライティング（ディレクショナル・ポイント・アンビエント）、3D→2Dラスタライゼーションパイプライン（2Dコンポジターとの統合）、基本マテリアル（ディフューズ・スペキュラー・エミッシブ）、wgpuによるGPUベース3Dレンダリングを実装する。

## 実装ステップ
1. テキストジオメトリからの3D押し出し・ベベル実装
2. 基本3Dプリミティブ実装（ボックス、球、シリンダー、平面）
3. 3Dカメラノード実装（透視投影/正射影）
4. 基本ライティング実装（ディレクショナル、ポイント、アンビエント）
5. 3D→2Dラスタライゼーションパイプライン実装（2Dコンポジターとの統合）
6. 基本マテリアル実装（ディフューズ、スペキュラー、エミッシブ）
7. wgpuによるGPUベース3Dレンダリング実装

## 対象コンポーネント
- `crates/ravel-3d/` (3Dレンダリングエンジン)
- `crates/ravel-3d/src/primitives/` (3Dプリミティブ)
- `crates/ravel-3d/src/camera/` (3Dカメラ)
- `crates/ravel-3d/src/lighting/` (ライティング)
- `crates/ravel-3d/src/material/` (マテリアル)
- `crates/ravel-gpu/src/render_3d/` (GPU 3Dレンダリング)
