# TASK-030: OFXプラグインプロセス分離実行
- **マイルストーン**: MS6 Pro Features
- **関連要件**: REQ-PLUGIN-001, REQ-PROJ-002, REQ-INFRA-007
- **規模**: L
- **依存タスク**: TASK-029

## 概要
OFXプラグイン実行用の子プロセス起動、IPCプロトコル（フレームデータ用共有メモリ）、プロセス間GPUメモリ共有、プラグインプロセスのクラッシュ検出・自動再起動、プラグイン障害時のグレースフルデグラデーション、プロセス分離によるメインプロセス保護を実装。サンプルOFXプラグインとの結合テストを行う。

## 実装ステップ
1. OFXプラグイン実行用子プロセス起動実装
2. IPCプロトコル実装（フレームデータ用共有メモリ）
3. プロセス間GPUメモリ共有実装
4. プラグインプロセスのクラッシュ検出・自動再起動実装
5. プラグイン障害時のグレースフルデグラデーション実装
6. セキュリティ: プロセス分離によるメインプロセス汚染防止の検証
7. サンプルOFXプラグインとの結合テスト

## 対象コンポーネント
- `crates/ravel-plugin/src/ofx/process/` (プロセス分離実行)
- `crates/ravel-plugin/src/ofx/ipc/` (IPCプロトコル)
- `crates/ravel-plugin/src/ofx/gpu_sharing/` (GPUメモリ共有)
- `crates/ravel-plugin/src/ofx/watchdog/` (クラッシュ検出・再起動)
