# TASK-032: WGSLカスタムシェーダノード
- **マイルストーン**: MS6 Pro Features
- **関連要件**: REQ-GPU-003
- **規模**: M
- **依存タスク**: TASK-005, TASK-014

## 概要
ShaderNodeタイプ（WGSLソース、入出力テクスチャ、ユニフォーム）、ランタイムWGSLコンパイル（エラーレポート付き）、シェーダエディタパネル（GPUIシンタックスハイライト付き）、ユニフォームパラメータ宣言（→アニメーションチャネル接続）、テクスチャ入出力バインディング、編集中のシェーダホットリロード、サンプルシェーダ（plasma、glitch、displacement）を実装する。

## 実装ステップ
1. ShaderNodeタイプ実装（WGSLソース、入出力テクスチャ、ユニフォーム）
2. ランタイムWGSLコンパイル実装（エラーレポート付き）
3. シェーダエディタパネル実装（GPUI、シンタックスハイライト付き）
4. ユニフォームパラメータ宣言実装（→アニメーションチャネル接続）
5. テクスチャ入出力バインディング実装
6. 編集中のシェーダホットリロード実装
7. サンプルシェーダ実装（plasma、glitch、displacement）

## 対象コンポーネント
- `crates/ravel-core/src/nodes/shader/` (シェーダノード)
- `crates/ravel-gpu/src/shader_runtime/` (WGSLランタイムコンパイル)
- `crates/ravel-ui/src/panels/shader_editor/` (シェーダエディタパネル)
