# TASK-033: テーマシステム + アクセシビリティ
- **マイルストーン**: MS7 Polish
- **関連要件**: REQ-UI-006
- **規模**: M
- **依存タスク**: TASK-006

## 概要
テーマ定義フォーマット（TOML: 色、ノードタイプ色、スコープ色）、ビルトインテーマ（Dark、Light）、色覚多様性テーマ（1型色覚・2型色覚・3型色覚）、外部TOMLファイルからのテーマ読み込み、テーマ切り替えUI、DPI/UIスケーリング、全パネルのキーボードオンリーナビゲーション、アニメーション軽減モード、テキスト要素のGPUIアクセシビリティツリー統合を実装する。

## 実装ステップ
1. テーマ定義フォーマット実装（TOML: 色、ノードタイプ色、スコープ色）
2. ビルトインテーマ実装（Dark、Light）
3. 色覚多様性テーマ実装（1型色覚 protanopia、2型色覚 deuteranopia、3型色覚 tritanopia）
4. 外部TOMLファイルからのテーマ読み込み実装
5. テーマ切り替えUI実装
6. DPI/UIスケーリング実装
7. 全パネルのキーボードオンリーナビゲーション実装
8. アニメーション軽減モード実装
9. テキスト要素のGPUIアクセシビリティツリー統合

## 対象コンポーネント
- `crates/ravel-ui/src/theme/` (テーマシステム)
- `crates/ravel-ui/src/theme/builtin/` (ビルトインテーマ)
