# TASK-043: シェイプのジオメトリ化 + インスタンス複製ノード
- **マイルストーン**: MS5 Motion Graphics
- **関連要件**: REQ-MOGRAPH-001 (v2)
- **規模**: L
- **依存タスク**: TASK-038, TASK-042
- **備考**: TASK-024（旧リピーター案）を v2 要件に合わせて置き換える

## 概要
シェイプ生成ノード（矩形/楕円/多角形/スター/カスタムパス）を `Geometry`
出力に実装（既存 FrameBuffer 直描き実装からの移行を含む）。複製ノード群
（グリッド配列/円形配列/パス沿い配列/ポイント散布）はインスタンス群を生成し、
`index`/`P`/`rot`/`scale` 標準属性を必ず付与する。

## 実装ステップ
1. シェイプ生成ノードの Geometry 出力化（パスプリミティブ生成）
2. グリッド/円形配列ノード（行列数・半径・角度範囲、index/P 付与）
3. パス沿い配列（弧長等間隔/個数指定、接線から rot 算出）
4. ポイント散布（面積内ランダム、seed 決定性、密度）
5. インスタンスのソース差し替え入力（任意ジオメトリを複製元にできる）
6. Node Editor のポート型表示・接続判定の対応
7. 統合テスト（属性付与、決定性、複製数）

## 対象コンポーネント
- `crates/ravel-nodes/src/shape/`, `crates/ravel-nodes/src/scatter/`
- `crates/ravel-core/src/registry/builtin.rs`
- `crates/ravel-app/src/panels/node_editor.rs`（ポート型対応）

## 完了条件
- [ ] 基本シェイプがジオメトリとして生成される
- [ ] 4種の複製で各インスタンスに index/P/rot/scale が付与される
- [ ] 任意ジオメトリ（テキスト変換含む）を複製ソースにできる
- [ ] 散布が seed 決定的
