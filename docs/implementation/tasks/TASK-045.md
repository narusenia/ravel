# TASK-045: パーティクル v2（ポイントジオメトリシミュレーション）
- **マイルストーン**: MS5 Motion Graphics
- **関連要件**: REQ-MOGRAPH-002 (v2)
- **規模**: L
- **依存タスク**: TASK-040, TASK-041, TASK-042
- **備考**: TASK-025（旧パーティクル案）を v2 要件に合わせて置き換える

## 概要
属性付きポイントジオメトリのフレーム逐次シミュレーションとしてパーティクル
を実装する。エミッタ（ポイント/ライン/エリア/任意ジオメトリ）、フィールド
由来フォース（重力/アトラクタ/ドラッグ/ノイズ）、寿命・サイズ・色の
per-point 属性カーブ、`id` 安定性、sim キャッシュ経由のスクラブ耐性を含む。
出力はポイントジオメトリであり、描画（TASK-042）や複製ソース（TASK-043）に
接続できる。

## 実装ステップ
1. `StatefulProcessor` 実装（状態 = ポイントジオメトリ + RNG 状態）
2. エミッタ（形状別発生、`id`/`age`/`velocity` 付与、seed 決定性）
3. フォース統合（`Field` を加速度として積分、複数フィールド合成）
4. 寿命管理（age/life、消滅、カーブによる pscale/Cd/alpha 変調）
5. 描画接続（ポイントスプライト）と複製ソース接続の統合テスト
6. スクラブ/巻き戻し/上流編集の sim キャッシュ挙動テスト
7. 性能測定（10万ポイント目標、rayon 並列積分）

## 対象コンポーネント
- `crates/ravel-nodes/src/particle/`
- `crates/ravel-core/src/geometry/sim.rs`

- [ ] スクラブ・巻き戻しが破綻しない（sim キャッシュ）
