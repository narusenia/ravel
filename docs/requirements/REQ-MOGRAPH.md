# REQ-MOGRAPH — モーショングラフィックス

## REQ-MOGRAPH-001: 基本シェイプ + インスタンス複製 + per-instance 変調

- **優先度**: Must
- **ステータス**: Revised (v2 — 属性システム統合)
- **説明**: 2Dシェイプ生成ノード（矩形、楕円、多角形、スター、カスタムパス）はジオメトリ（REQ-CORE-010）を出力する。配列複製ノード（グリッド配列、円形配列、パス沿い配列、ポイント散布）は**インスタンス群**を生成し、各インスタンスに標準属性（`index` / `P` / 回転 / スケール）を付与する。インスタンスごとのパラメータ変調（位置オフセット、スケール、色、時間オフセット等）は per-instance 属性 + フィールド（REQ-CORE-012）+ Lua 式で行い、Cavalry の duplicator + falloff 相当の自由度を提供する。全パラメータが統一アニメーションチャネル（REQ-CORE-007）で数式ドリブン可能。最終描画はジオメトリ→フレームバッファのラスタライズノードが担う。
- **受入条件**:
  - [ ] 基本シェイプ（矩形/楕円/多角形/スター）がジオメトリとして生成される
  - [ ] カスタムパスでシェイプを作成できる
  - [ ] グリッド/円形/パス沿い配列で複製でき、各インスタンスが index/P 属性を持つ
  - [ ] インスタンス属性をフィールド/式で変調してパラメータに反映できる
  - [ ] 変調結果（例: 距離フォールオフでスケールが波打つグリッド）が動作する
  - [ ] ジオメトリのラスタライズ結果が合成パイプラインに流れる
  - [ ] パラメータにLua式を設定できる
- **依存**: REQ-CORE-003, REQ-CORE-007, REQ-CORE-010, REQ-CORE-012

## REQ-MOGRAPH-002: パーティクル（ポイントジオメトリシミュレーション）

- **優先度**: Should
- **ステータス**: Revised (v2 — ジオメトリ/ステートフル評価統合)
- **説明**: パーティクルは**属性付きポイントジオメトリ**（REQ-CORE-010）のフレーム逐次シミュレーション（REQ-CORE-011）として実装する（v1 の「2Dフレームバッファ上の処理に収まる」制約は撤回 — REQ-MOGRAPH-004 のジオメトリ相互作用と両立しないため）。エミッタノード（ポイント/ライン/エリア/任意ジオメトリ発生源、テキスト変換ジオメトリを含む）。フォースは汎用フィールド（REQ-CORE-012: 重力、アトラクタ、ドラッグ、ノイズ、タービュランス）を接続する。寿命・サイズ・色・不透明度は per-point 属性としてカーブ/フィールドで制御。描画はポイントラスタライズ/スプライトノードが担い、パーティクルを他のジオメトリ処理（インスタンス複製のソース等）にも流用できる。
- **受入条件**:
  - [ ] パーティクルが属性付きポイントジオメトリとして発生・更新される
  - [ ] エミッタ形状（ポイント/ライン/エリア/任意ジオメトリ）を選択できる
  - [ ] フィールド由来のフォース（重力/アトラクタ/ドラッグ/ノイズ)が影響する
  - [ ] 寿命/サイズ/色が per-point 属性としてカーブ制御できる
  - [ ] スクラブ・巻き戻しがシミュレーションキャッシュで破綻しない
  - [ ] パーティクルポイントを他ノード（インスタンス複製等）のソースに使える
- **依存**: REQ-CORE-010, REQ-CORE-011, REQ-CORE-012, REQ-GPU-001

## REQ-MOGRAPH-003: 3D基本機能

- **優先度**: Should
- **ステータス**: Draft
- **説明**: 2Dコンポジット内に3D要素を配置するElement 3D/Cinema 4D Lite的スコープ。3Dテキスト押し出し、基本プリミティブ（ボックス、球、円柱、平面）、カメラ（パース/オルソ）、基本ライティング（ディレクショナル/ポイント/アンビエント）。フル3Dシーンは目指さない。リリックモーションでの3Dテキスト多用が主要ユースケース。
- **受入条件**:
  - [ ] 3Dテキストの押し出し・ベベルが可能
  - [ ] 基本3Dプリミティブを配置できる
  - [ ] 3Dカメラ（パース/オルソ）で投影できる
  - [ ] 基本ライティングが機能する
  - [ ] 3Dレンダリング結果が2Dコンポジットに合成される
- **依存**: REQ-MOGRAPH-004, REQ-GPU-001

## REQ-MOGRAPH-004: プロシージャルタイポグラフィ

- **優先度**: Must
- **ステータス**: Draft
- **説明**: テキストアニメータ（文字単位のイン/アウト、ウェーブ、ランダム等）。パスに沿ったテキスト配置。テキスト→ジオメトリ（パス/メッシュ）変換。変換後のジオメトリはフィールド/パーティクル（REQ-MOGRAPH-002）の影響を受けられる（文字が粒子になって散る、ノイズフィールドで歪む等）。3Dテキスト押し出し（REQ-MOGRAPH-003）。フォントパース/ラスタライズにswash + zenoを活用。テキストアニメーションはノードグラフで構築可能。
- **受入条件**:
  - [ ] テキストレイヤーを作成できる
  - [ ] フォント/サイズ/色/行間等の基本属性を設定できる
  - [ ] 文字単位アニメーション（イン/アウト/ウェーブ/ランダム）が動作する
  - [ ] パスに沿ったテキスト配置ができる
  - [ ] テキストをパスジオメトリに変換できる
  - [ ] 変換ジオメトリがフィールド/パーティクルの影響を受ける
  - [ ] テキストアニメーションをノードグラフで構築できる
- **依存**: REQ-CORE-001, REQ-CORE-007, REQ-CORE-010, REQ-CORE-012

## REQ-MOGRAPH-005: ビルトインエフェクトライブラリ

- **優先度**: Must
- **ステータス**: Draft
- **説明**: 出荷時に十分なビルトインノード/エフェクトを同梱し、外部プラグインなしで基本作業が完結すること。
- **カテゴリ別ノード一覧**:
  - **合成**: ブレンドモード（加算/乗算/スクリーン/オーバーレイ等）、マスク、キーイング
  - **カラー調整**: 明るさ/コントラスト、色相/彩度、カーブ、レベル
  - **カラーグレーディング**: リフト/ガンマ/ゲイン、カラーホイール、HSLカーブ、LUT適用、カラーマッチ
  - **ブラー/シャープ**: ガウシアンブラー、方向ブラー、ラジアルブラー、シャープ
  - **ディストーション**: ワープ、レンズ歪み、波紋
  - **生成**: グラデーション、ノイズ、フラクタル、チェッカーボード
  - **スタイライズ**: グロウ、シャドウ、ストローク、エンボス
  - **トランスフォーム**: 移動、回転、スケール、ミラー、タイル
  - **トランジション**: クロスディゾルブ、ワイプ、プッシュ、スライド等20種程度
  - **時間**: タイムリマップ、フレームブレンド、フリーズ
- **受入条件**:
  - [ ] 上記カテゴリの主要ノードが全て動作する
  - [ ] トランジションがタイムラインUIから適用できる
  - [ ] 全ノードのパラメータが統一アニメーションチャネルに対応する
- **依存**: REQ-CORE-002, REQ-GPU-001
