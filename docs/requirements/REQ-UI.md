# REQ-UI — ユーザーインターフェース

## REQ-UI-001: GPUIフレームワーク採用

- **優先度**: Must
- **ステータス**: Draft
- **説明**: Zed由来のGPUIをUIフレームワークとして採用。カスタムビューポート（タイムライン、ノードグラフ、ビューア）の自由な描画が可能。wgpuベースのため処理エンジンとGPUコンテキスト共有が自然。テキスト描画はswash + zenoを活用。
- **受入条件**:
  - [ ] GPUIでアプリケーションウィンドウが表示される
  - [ ] カスタムビューの描画が可能
  - [ ] テキスト入力・IMEが動作する
  - [ ] macOS / Windowsで動作する

## REQ-UI-002: ノードグラフエディタ

- **優先度**: Must
- **ステータス**: Draft
- **説明**: 2Dキャンバス上にノードを自由配置しワイヤーで接続。スナップガイド・自動整列補助（手動トリガー）を提供。サブグラフ（グループノード）による階層化で大規模グラフを管理可能。プリセット/テンプレートとしてサブグラフを保存・再利用可能。
- **受入条件**:
  - [ ] ノードの追加・削除・移動・接続が可能
  - [ ] ワイヤーのドラッグ接続が可能
  - [ ] スナップガイドが表示される
  - [ ] 自動整列機能が動作する
  - [ ] ノード群をサブグラフにグループ化できる
  - [ ] サブグラフの公開パラメータを定義できる
  - [ ] 1000ノード規模で操作が快適に行える
- **依存**: REQ-UI-001, REQ-CORE-001

## REQ-UI-003: リッチタイムライン + ノードグラフ展開

- **優先度**: Must
- **ステータス**: Draft
- **説明**: タイムライン上でクリップ配置、トランジション、エフェクトスタック積み上げ、キーフレーム打ちが可能。エフェクトスタックをワンクリックでノードグラフビューに展開でき、自由な接続編集が可能。内部表現は常にノードグラフで、タイムラインUIは「制約付きビュー」として機能。エフェクトスタックが直列チェーンの間はタイムライン表示可能、分岐/マージ時にノードグラフ表示を促す。
- **受入条件**:
  - [ ] クリップの追加・削除・移動・トリミングが可能
  - [ ] クリップにエフェクトスタックを積める
  - [ ] エフェクトパラメータのキーフレームをタイムライン上で打てる
  - [ ] トランジション（クロスディゾルブ等）を適用できる
  - [ ] エフェクトスタックをノードグラフに展開できる
  - [ ] ノードグラフでの変更がタイムライン表示に反映される
- **依存**: REQ-UI-001, REQ-CORE-001

## REQ-UI-004: スコープ付きビューア + パネルトグル

- **優先度**: Must
- **ステータス**: Draft
- **説明**: ノードグラフの任意ノードの出力をプレビュー表示。再生/停止、フレーム送り、ズーム、フィット表示。波形モニタ、ベクトルスコープ、ヒストグラム、パレードの各スコープをサポート。OCIOカラースペースを反映。各パネル（ビュー/波形/ベクトルスコープ/ヒストグラム/パレード）はメニュー「表示」からトグル表示。将来マルチビューア（A/B比較、差分表示）および外部モニタ出力（DeckLink等）に対応。
- **受入条件**:
  - [ ] 任意ノードの出力をビューアに表示できる
  - [ ] 再生/停止/フレーム送りが動作する
  - [ ] 波形モニタが表示される
  - [ ] ベクトルスコープが表示される
  - [ ] ヒストグラムが表示される
  - [ ] パレードが表示される
  - [ ] メニューから各スコープパネルをトグルできる
  - [ ] OCIOカラースペースがビューア表示に反映される
- **依存**: REQ-RENDER-003

## REQ-UI-005: ワークスペースプリセット + カスタマイズ

- **優先度**: Must
- **ステータス**: Draft
- **説明**: デフォルトのワークスペースプリセット（Edit/Node/Color/Motion）を提供。パネルの追加/削除/リサイズ/並べ替えが可能。カスタムワークスペースを保存可能。`gpui_component`のsheet/dock機能を基盤に構築。段階的にフリードッキング（全パネル自由ドッキング/フローティング/タブ化）へ拡張。
- **デフォルトプリセット**:
  - **Edit**: タイムライン + ビューア + メディアビン + プロパティ
  - **Node**: ノードグラフ + ビューア + プロパティ + カーブエディタ
  - **Color**: ビューア + スコープ群 + ノードグラフ（カラコレチェーン）
  - **Motion**: ノードグラフ + ビューア + カーブエディタ + テキストエディタ
- **受入条件**:
  - [ ] デフォルトプリセットで起動できる
  - [ ] プリセットを切り替えられる
  - [ ] パネルの追加/削除/リサイズが可能
  - [ ] カスタムワークスペースを保存/復元できる
- **依存**: REQ-UI-001

## REQ-UI-006: テーマシステム + アクセシビリティ

- **優先度**: Should
- **ステータス**: Draft
- **説明**: ダーク/ライトテーマ + カスタムテーマ。テーマ定義はJSON/TOMLで外部化しコミュニティ共有可能。色覚多様性対応テーマ（第1/第2/第3色覚）。UIスケーリング（DPI対応）。アニメーション削減モード。キーボード完全操作対応。テキスト要素はGPUIアクセシビリティツリーに公開。
- **受入条件**:
  - [ ] ダーク/ライトテーマの切り替えが可能
  - [ ] カスタムテーマをJSON/TOMLから読み込める
  - [ ] 色覚多様性テーマが選択できる
  - [ ] UIスケーリングが設定できる
  - [ ] 全操作がキーボードで完結する
  - [ ] アニメーション削減モードが機能する
- **依存**: REQ-UI-001

## REQ-UI-008: メディアビン + メタデータ

  - [ ] サムネイルプレビューが表示される
  - [ ] メタデータ（解像度/コーデック/FPS/長さ）が表示される
  - [ ] 検索・フィルタリングが機能する

## REQ-UI-009: マルチモニタ + デタッチ + 専用ビューアウィンドウ

- **優先度**: Should

## REQ-UI-010: ノード/クリップコピペ + ファイルD&D

- **優先度**: Must
- **ステータス**: Draft
- **説明**: ノードグラフ内のノード群やタイムラインクリップをコピー/ペースト。コピーデータは内部形式（RON/JSON）でクリップボードに保持し、別Ravelインスタンス間でも移動可能。Finder/Explorerからのメディアファイルドラッグ&ドロップでインポート。将来的に外部アプリとの連携プロトコル対応。
- **受入条件**:
  - [ ] ノードのコピー/ペーストが動作する
  - [ ] クリップのコピー/ペーストが動作する
  - [ ] 別Ravelインスタンスへのペーストが動作する
  - [ ] OSからのファイルD&Dでメディアがインポートされる
